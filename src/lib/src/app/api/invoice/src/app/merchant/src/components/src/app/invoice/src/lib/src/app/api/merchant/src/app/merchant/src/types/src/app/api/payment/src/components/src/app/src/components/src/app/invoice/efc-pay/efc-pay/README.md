# Efikcoin EFC Pay

Efikcoin merchant payment application built on BNB Smart Chain.

## Token Information

Name: Efikcoin

Symbol: EFC

Network: BNB Smart Chain

Contract:

0x677ce9cba67f7484ea951a12897ce780cfd8fed1

## Features

- Merchant registration
- EFC payment invoices
- EFC payment information
- Merchant dashboard foundation
- Blockchain payment integration foundation

## Technology

- Next.js
- TypeScript
- BNB Smart Chain

## Development

Install dependencies:

npm install

Run development server:

npm run dev
